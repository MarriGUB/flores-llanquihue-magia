# Links

Este archivo contiene dos enlaces útiles para ver la aplicación y pasos rápidos para usarlos.

- Enlaces al ejecutar localmente (dev server):

- Local (solo en esta máquina): http://localhost:8080/
  - Qué hace: abre la aplicación en el navegador del equipo donde se ejecuta el servidor de desarrollo.
  - Uso: abrir en el navegador del mismo equipo.

- Red (desde otros dispositivos en la misma LAN): http://192.168.1.13:8080/
  - Qué hace: permite acceder a la app desde otros dispositivos de tu red local (móvil, otra PC).
  - Uso: abrir en el navegador de otro dispositivo conectado a la misma red.

- Enlace local (desarrollo): http://localhost:5173
  - Qué hace: muestra la aplicación que se ejecuta en tu máquina local.
  - Pasos rápidos:
    1. Instalar dependencias si no están: `npm install`
    2. Iniciar servidor de desarrollo: `npm run dev`
    3. Abrir en el navegador: `http://localhost:5173`

Notas:
- Ajusta el puerto si tu proyecto usa otro.
- La URL del repositorio es https://github.com/MarriGUB/flores-llanquihue-magia (no es la página desplegada).
